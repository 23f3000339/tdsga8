Revenue Growth Analysis

Year-over-Year Growth Rate:

\[ \text{Growth Rate} = \frac{\text{Revenue}_{\text{current}} - \text{Revenue}_{\text{previous}}}{\text{Revenue}_{\text{previous}}} \times 100\% \]

Return on Investment (ROI):

\[ \text{ROI} = \frac{\text{Net Profit}}{\text{Total Investment}} \times 100\% = 14.2\% \]

Compound Annual Growth Rate (CAGR):

\[ \text{CAGR} = \left(\frac{\text{Ending Value}}{\text{Beginning Value}}\right)^{\frac{1}{n}} - 1 \]
These formulas demonstrate our analytical approach to measuring performance. Our 14.2% ROI exceeds industry average by 3.5 percentage points.

import pandas as pd
import numpy as np

# Calculate portfolio returns
def calculate_portfolio_return(assets, weights, returns):
    portfolio_return = np.sum(weights * returns)
    return portfolio_return

# Risk-adjusted performance (Sharpe Ratio)
def sharpe_ratio(returns, risk_free_rate=0.02):
    excess_return = returns - risk_free_rate
    return excess_return / np.std(returns)

# Example usage
portfolio_data = pd.read_csv('quarterly_data.csv')
performance = calculate_portfolio_return(
    portfolio_data['assets'], 
    portfolio_data['weights'], 
    portfolio_data['returns']
)
